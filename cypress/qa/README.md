# 📊 Gestão de Qualidade com Jira

Este projeto inclui simulação de atividades de QA utilizando Jira (Kanban), com foco em gerenciamento de bugs e organização de tarefas.

## 🛠️ Atividades realizadas

- Criação e registro de bugs
- Escrita de cenários de teste
- Acompanhamento do ciclo de vida dos defeitos
- Organização de tarefas em board Kanban

## 🐞 Exemplos de Bugs

### Bug: Campo nome permite envio vazio

**Passos para reproduzir:**
1. Acessar o formulário
2. Deixar o campo nome vazio
3. Clicar em enviar

**Resultado esperado:**
Exibir mensagem de erro

**Resultado obtido:**
Formulário enviado com campo vazio
